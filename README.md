# ☕ Coffee Landing Page

A modern and responsive **Coffee Landing Page** built using **HTML5** and **CSS3**. This project is designed to provide an elegant and visually appealing landing page for a coffee shop or café. It focuses on clean design, responsive layout, and an engaging user experience while following front-end development best practices.

---

## 🚀 Features

* Modern and responsive user interface
* Attractive hero section with call-to-action
* Clean and organized navigation bar
* About section for business information
* Coffee menu/products showcase
* Responsive layout for desktop, tablet, and mobile devices
* Well-structured HTML and CSS code
* Simple and easy-to-understand project structure

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**




## 🎯 Purpose of the Project

This project was created to strengthen front-end development skills by practicing:

* Semantic HTML5
* CSS3 Styling
* Responsive Web Design
* Flexbox Layout
* UI Design Principles
* Clean Code Structure
